# 第一章进入新时代

## 我国社会主要矛盾转化的依据

我国社会生产力水平总体上显著提高

1. 经过改革开放的深入推进，**我国社会生产力水平总体上显著提高**，社会生产力再很多方面进入世界前列。
2. 人民生活水平显著提高，任命对美好生活的向往更加强烈。

1. 没有改变我们对我国社会主义所处历史阶段的判断。
