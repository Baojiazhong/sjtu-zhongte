## 完善全球生态治理体系的几个举措（p220）

1. 秉持人类命运共同体理念、积极参与全球生态治理，为全球提供更多公共产品，展现我国负责任大国形象
2. 加强南南合作，以及同周边国家合作，为发展中国家提供力所能及的资金、技术支持、帮助环境治理能力，共同打造绿色”一带一路”。
3. 坚持共同但有区别的责任原则、公平原则和各自能力原则，坚定维护多边主义，坚决维护我国发展利益。
